# 💼 Job Market & Salary Analytics Dashboard

An interactive data analytics web application built using Python and Streamlit that analyzes real-world job market trends, salaries, skills demand, and employment patterns.

---

## 🚀 Project Overview

This project explores job market data (2022–2026) and provides actionable insights through an interactive dashboard.

It helps users understand:

- 💰 Average salary by job title
- 🌍 Salary comparison across countries
- 🧠 Most in-demand skills
- 🏢 Company type impact (Startup vs Big Tech)
- 💻 Remote vs Onsite job trends
- 📈 Experience level salary distribution

---

## 🧰 Tech Stack

- Python  
- Pandas  
- NumPy  
- Plotly  
- Streamlit  

---

## 📁 Dataset Information

The dataset includes the following columns:

- Year  
- Job_Title  
- Country  
- Company_Type  
- Experience_Level  
- Salary_USD  
- Remote  
- Top_Skill  

---

## 📊 Features

### 🔹 Data Analysis
- Salary trends by job title
- Country-wise salary comparison
- Skill demand frequency
- Experience level analysis

### 🔹 Interactive Dashboard
- Sidebar filters (Job, Country, Experience, Remote)
- Dynamic charts
- Real-time data updates
